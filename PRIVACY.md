# Privacy Policy

**Last Updated:** 2026-02-07

**Skills Manager** is designed as a local-first desktop application. We respect your privacy and do not collect your personal data.

## 1. No Data Collection
We do not collect, store, transmit, or share any of your:
- Personal Information
- Usage Data or Analytics
- Skills content or configurations
- File system information

## 2. Local Data Storage
All data generated or used by Skills Manager stays on your local device.
- **Configuration:** Stored in `~/.skills-manager/config.json` (or your OS equivalent).
- **Skills:** Stored in the directories you configure.

## 3. Network Usage
The application operates entirely offline. It does not communicate with any backend servers to track usage.
*Note: If you download updates via GitHub Releases, standard GitHub interactions apply.*

## 4. Third-Party Tools
Skills Manager manages files for third-party tools (e.g., Claude Code, Codex). Please refer to their respective privacy policies regarding how they handle data when you use them.

## 5. Contact
If you have questions, please open an issue on our [GitHub Repository](../../issues).
