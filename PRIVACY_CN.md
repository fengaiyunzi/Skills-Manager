# 隐私政策 (Privacy Policy)

**最后更新：** 2026-02-07

**Skills Manager** 是一款本地优先的桌面应用程序。我们尊重您的隐私，承诺不收集您的任何个人数据。

## 1. 不收集数据
我们**不会**收集、存储、传输或分享您的：
- 个人信息
- 使用数据或行为统计
- Skills 内容或配置信息
- 文件系统信息

## 2. 本地数据存储
Skills Manager 产生或使用的所有数据均保留在您的本地设备上。
- **配置文件**：存储在 `~/.skills-manager/config.json`（或您操作系统的对应位置）。
- **Skills 数据**：存储在您配置的本地目录中。

## 3. 网络使用
本应用程序完全离线运行。它不会与任何后台服务器通信以跟踪使用情况。
*注：如果您通过 GitHub Releases 下载更新，将适用 GitHub 的标准网络交互规则。*

## 4. 第三方工具
Skills Manager 用于管理第三方工具（如 Claude Code、Codex）的文件。关于这些工具在运行时如何处理数据，请参阅它们各自的隐私政策。

## 5. 联系我们
如有疑问，请在我们的 [GitHub 仓库](../../issues) 中提交 Issue。
